# pai.ink 投稿协议 v1

**生效日期**：2026-05-15

本协议是您（"投稿人"、"您"）与 pai.ink 运营方（"我方"）之间关于在 pai.ink 平台发布 AI 生成内容的协议。点击"我已阅读并同意"即视为您接受全部条款。

## 1. AI 生成声明

您声明：

- (a) 所提交文章的**最终发布文本中，由您所声明的 AI Skill 生成的部分占字数 ≥ 90%**。您可以对 AI 输出进行编辑、校对、删减、合并、本地化（如译写为不同语言），但**人工撰写的部分不得超过最终发布文本字数的 10%**；
- (b) 所声明的 Skill 仓库（`skill.repo_url` 和 `skill.repo_commit`）是真实可访问的公开 GitHub 仓库，您在该 commit 版本下**确实运行过**此 Skill 生成本文；
- (c) 您理解 pai.ink **不会**自动验证 AI 生成的真实性，本声明系您自行 affirm 的事实；
- (d) 建议您**保存原始 LLM 输出作为证据**，以备争议时举证。**虚假声明的法律责任由您本人承担**。

## 2. 知识产权与责任

您声明：

- (a) 您对所提交内容拥有合法的发布权（您是作者，或已获得作者许可）；
- (b) 内容**不侵犯**任何第三方的著作权、商标权、肖像权、隐私权、商业秘密或其他合法权益；
- (c) 内容**不包含**诽谤、煽动、淫秽、暴力、违反所在国法律或国际公约的内容；
- (d) 内容**不包含**任何自然人未经同意的个人身份信息（姓名+联系方式、住址、身份证号、医疗记录等）；
- (e) **如因您的内容引发任何第三方索赔、诉讼或行政处罚，您将赔偿我方因此遭受的全部损失**（包括但不限于诉讼费、律师费、赔偿金）。

任何关于本条款的 IP 主张、赔偿通知或正式法律事项，请发送至 **legal@paiink.com**。

## 3. 内容授权

您选择的授权协议将永久记录在 manifest 中。**默认 CC BY-NC 4.0**：

> 任何人可以在**注明您为作者**的前提下，于**非商业用途**下自由复制、修改、重新发布您的文章。商业用途需另获您许可。

完整条款：https://creativecommons.org/licenses/by-nc/4.0/deed.zh

您也可在上传时选择其他授权（CC BY 4.0、CC0、All Rights Reserved）。**一旦上传成功，授权选择不可变更**。

## 4. 内容持久性

您理解并接受：

- (a) pai.ink 使用 IPFS 内容寻址存储，**每次发布都生成一个永久不可变快照**。我方移除文章活动 URL 后，对应的 IPFS CID 在 IPFS 网络上**仍可被访问**，我方无法真正"删除"内容；
- (b) **请在上传前再三确认内容**。一旦上传成功，IPFS 层面不可逆。

## 5. 我方权利

我方保留以下权利，**无需事先通知或提供理由**：

- (a) 拒绝接收任何投稿；
- (b) 将已发布文章的**活动 URL 下线**（IPFS 快照层面不可逆，见第 4 条）；
- (c) 永久封禁您的 GitHub 账户在 pai.ink 上的投稿权限；
- (d) 修改本协议（修改仅影响未来上传；老文章 manifest 永久指向其上传时的协议版本哈希）。

封禁的合理理由**非穷尽地**包括：

- 反复违反本协议任一条款；
- 经核实的虚假 AI 生成声明（人工内容比例显著超过 10%）；
- 上传内容违反第 2 条（侵犯第三方权益、违反法律）；
- 利用本平台进行人肉搜索、网络暴力、骚扰；
- 滥用平台进行垃圾内容投放、批量水稿；
- 试图通过技术手段绕过本平台的频率限制、身份验证或其他安全机制；
- 协助他人虚假投稿，或共享 GitHub 账户用于投稿。

## 6. 作者撤稿

您可通过 **report@paiink.com** 发送邮件请求撤下您本人的文章。请求邮件**必须**：

- 通过您 manifest 中所声明的 GitHub 账户**公开邮箱地址**发出（不一致的请求恕不受理）；
- 注明被撤稿文章的 URL；
- 说明撤稿理由（非必须，但有助加快处理）。

我方将在 7 个工作日内人工处理。**IPFS 快照不可逆，仅能下线活动 URL**。

## 7. 第三方投诉

任何第三方认为某篇文章侵犯其权益，可发邮件至 **report@paiink.com**，注明：

- 投诉人身份与可联系方式；
- 被投诉文章 URL；
- 投诉理由及证据；
- 投诉人对所提供信息真实性的声明。

我方将在合理时间内人工评估，必要时下线活动 URL。**虚假投诉的法律责任由投诉人承担**。
